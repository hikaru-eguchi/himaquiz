# 進撃の巨人ファン必見！クイズ15問で真の知識を試そう

## 問題1

<Answer>
普通の巨人  
エレンが初めて巨人化した時に戦ったのは、調査兵団の仲間を救うために現れた普通の巨人です。この場面でエレンの巨人化能力が初めて明かされ、物語は大きく進展します。
</Answer>

## 問題2

エルヴィン・スミスは調査兵団の団長であり、数々の作戦で人類を巨人から守るための戦略を立てました。彼のリーダーシップは多くの兵士から信頼されています。
