# 【初対面でも話せる】合コンで盛り上がるクイズ20選

最初がちょっと難しい。

何を話せばいいかわからない。

でも無言も気まずい。

そんな時に使いやすいのが、

**合コン向けクイズ**

です。

クイズといっても、
ガチの知識問題だけではありません。

- 恋愛2択

- 軽いひっかけ

- 飲み会で使いやすいネタ

このあたりを混ぜると、
かなり盛り上がります。

今回は、

**合コン・飲み会・通話で使いやすいクイズ20問**

初対面。

### 盛り上がるポイント

最初から会話しすぎなくていいのが楽なタイプ。

水族館派は雰囲気重視。

居酒屋派は、
自然に話せる空気が好きなタイプ。

初対面でもかなり使いやすいです。

「なんでそれ選んだの？」

- 初対面
